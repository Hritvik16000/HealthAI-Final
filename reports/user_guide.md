# HealthAI User Guide

## Run locally
1. Activate the environment.
2. Run preprocessing and training scripts.
3. Start FastAPI.
4. Start Streamlit.

## Features
- Risk prediction
- Length of stay prediction
- Patient clustering
- Association rules
- Sentiment analysis
- RAG chatbot
- Translation
- Advanced demo modules: MLP, LSTM, CNN, ClinicalBERT, SHAP, MLflow

## Disclaimer
This project is a course-style demonstrator and not a clinical decision system.
